# Project Summary — Bente Analytics Dashboard

## Project Overview

Bente Analytics is a public safety operational intelligence platform designed to support crowd monitoring, hotspot detection, deployment planning, predictive forecasting, event monitoring, and incident investigation.

The solution is intended for pilot deployment with local Police Departments and hospitality partners.

---

## Project Objectives

The project was developed to:

* Visualize real-time crowd activity
* Identify high-risk venues and hotspots
* Support officer deployment decisions
* Forecast future crowd activity
* Identify events driving crowd density
* Provide investigation tools for venue-related incidents
* Provide a natural-language operational assistant
* Deliver actionable operational intelligence

---

## Deliverables

### Home / Officer Landing Page

Operational starting point for officers at the beginning of a shift.

Features include:

* Officer greeting and jurisdiction summary
* Live shift status and current time
* Operational summary (critical/high-risk venues, zone alerts)
* Quick navigation to all dashboard modules

### Real-Time Crowd Heatmap

Interactive heatmap displaying crowd concentration and venue activity.

Features include:

* Risk-based venue markers
* Active incident monitoring
* Operational search
* Venue investigation access

### Peak-Hour Analysis

Identification of:

* Peak risk periods
* Highest-risk zones
* High-priority venue activity

### Predictive Forecasting

Forecasting models estimate future venue foot traffic using historical activity patterns.

Outputs include:

* Predicted crowd activity
* Expected hotspot locations
* Deployment planning guidance

### Events Monitoring

Identifies events contributing to elevated crowd density.

Features include:

* Event name, category, size, and time window
* Venue, city, county, state, and patrol zone
* Crowd density and risk level per event
* Geographic and risk-level filters
* Direct links to map and investigation

### Incident Investigation

Investigation tools provide:

* Venue presence lookup
* Historical activity review
* Crowd density timeline analysis
* User presence records
* Operational intelligence summaries
* Geographic filters — County, City

### AI Operational Assistant

Natural-language assistant for operational queries.

Features include:

* Plain-language questions about zones, counties, and venues
* Peak hour and risk-level summaries by jurisdiction
* Event-driven crowd density explanations
* Recommended officer actions based on live dataset

---

## Operational Design Philosophy

The dashboard is designed as a public safety command center rather than a traditional reporting platform.

Each component focuses on:

* Situational awareness
* Hotspot identification
* Risk assessment
* Resource allocation
* Officer action recommendations
* Event-driven crowd explanations
* Investigation support

The objective is to transform operational data into actionable decision support using plain language, with no technical jargon.
